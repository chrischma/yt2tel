# yt2tel
A  `youtube-dl`-based terminal script that converts any youtube-video to mp3 and sends it to you via telegram. For personal use only, obviously. 


## Usage

Run `python3 yt2tel` via terminal to run the script. paste your URL and that's it.

## Setup

As i am using the **telegram_send** package in this early version, please make sure to install it before running yt2tel.


To install telegram_send, run: 

`pip3 install telegram-send`
or
`sudo pip3 install telegram-send`


To configure telegram_send run: 

`telegram-send --configure`

You can find detailed instructions over here:
https://pypi.org/project/telegram-send/
